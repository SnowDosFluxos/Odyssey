# 🎮 RetroDownloads 2010 - Portal Retro de Downloads

Um site de downloads no estilo dos anos 2010 com design nostálgico, inspirado nos grandes portais de jogos, revistas de detonados e softwares daquela época.

## 📋 Características

✨ **Design Retro Autêntico:**
- Cores vibrantes (Azul #0066cc, Amarelo #ffcc00, Laranja)
- Layout clássico com cards e botões grandes
- Anúncios chamativos estilo 2010
- Totalmente responsivo

🎮 **Seções Principais:**
- **Home** - Destaques e principais downloads
- **Jogos** - 1000+ jogos categorizados (Ação, RPG, Estratégia, etc)
- **Revistas** - Edições de SuperGamePower, Nintendo Power, PlayStation Magazine
- **Softwares** - Utilitários, compactadores, leitores de mídia
- **Contato** - Formulário + FAQ retro

⚙️ **Funcionalidades:**
- Barra de busca interativa
- Contador de visitantes dinâmico
- Simulador de download com efeitos visuais
- Formulário de contato funcional
- Efeitos hover em cards e botões
